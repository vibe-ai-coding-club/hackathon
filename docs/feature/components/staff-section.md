# StaffSection

## 목적

랜딩 페이지 최하단에서 운영진 소개용 2줄 롤링 섹션을 제공한다.

## 동작/UI

- `SectionTitle(Staffs)`와 소개 문구를 중앙 정렬로 노출한다.
- 섹션은 진한 핑크 계열 그라디언트 배경과 white 텍스트를 사용한다.
- `StaffRoller`는 `w-screen` 풀블리드로 노출되어 화면 가로폭 전체를 채운다.
- 핑크 영역 내부 하단에도 별도 패딩을 두어 마지막 카드 줄이 답답하게 붙지 않도록 한다.
- Footer 및 하단 플로팅 스톱워치와 겹치지 않도록 섹션 하단에 white spacer 여백을 둔다.
- Footer 바로 위에 배치되는 랜딩 마지막 본문 섹션이다.

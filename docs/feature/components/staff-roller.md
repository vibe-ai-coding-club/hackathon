# StaffRoller

## 목적

운영진 목록을 위/아래 2줄의 반대 방향 무한 롤링으로 노출한다.

## Props

- `items: StaffItem[]`

## 동작/UI

- 전체 목록을 절반으로 나눠 위 줄/아래 줄에 배치한다. 홀수 개면 위 줄이 1개 더 가진다.
- 위 줄은 왼쪽 방향, 아래 줄은 오른쪽 방향으로 CSS keyframes 기반 무한 이동한다.
- 각 줄은 동일한 카드 목록을 여러 세트 반복 렌더링해, 데이터 수가 적어도 화면 전체에서 끊김 없이 롤링되도록 한다.
- PC: `hover` 동안 두 줄 모두 애니메이션 일시정지.
- Mobile: `touch start` 동안 일시정지, `touch end/cancel` 시 재시작.
